# Student Performance Data Analysis

## Objective
Analyze how test preparation courses and lunch type relate to student exam scores.

## Tools Used
- Python
- Pandas
- Matplotlib

## Workflow
Load → Inspect → Clean → Analyze → Visualize → Conclude

## Key Insights
- Students who completed the test preparation course scored higher on average.
- Reading and writing scores improved more than math scores.
- Students with standard lunch performed better in math on average.

## Dataset
Students Performance in Exams (Kaggle)
